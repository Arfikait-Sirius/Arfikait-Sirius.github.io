head と body

head - HTML のメタ情報を記述するタグ

タイトルの編集、メタデータ定義を行うことができる。
知識のあるエンジニアでもない限りさわる機会はほとんどない。
<title> タグでタイトルを編集することができる。

body - HTML で実際に画面に表示する情報を記述するタグ

これから主に編集するのはこの body タグ内になる。
以下に記載のある様々なタグは body タグ中で使用する。

エスケープ文字

エスケープ文字 - HTML の特殊文字を画面に表示するために使用する制御文字

"<" や ">" みたいな文字は HTML ファイルではタグの開始終了文字として認識されてしまい、
そのまま表示文字列として書いてもエラーになってしまう。
"<" みたいに画面に出すためには "&○○;" の形式で特殊文字のコードを書くと表示できる。
ちなみに "<" は "&lt;" で、">" は "&gt;" でエスケープできる。

id と class

id - そのタグを一意に特定する識別子

たくさんの章だったり枠だったりを識別子なしで並べると、
あとから色を付けたり動きを付けたりするときにどれに付ければいいかの指定が出来なくなってしまう。
そこで、ある程度まとまった章や枠には id を付けて管理する仕組みが用意されている。

class - そのタグをグルーピングする識別子

id はタグ 1つ 1つに対して付けていくのに対して、class はグループごとになるイメージ。
ある程度のまとまりで色や動きが同じ場合は、
class を使って「このグループはこの色」みたいに一括指定できる仕組みが用意されている。

form と input

form - サーバーに送信するデータをまとめるタグ

データを送信する場合、「どのタグのどの値を送信するか」を決めておかないと、
想定外の値まで送信されてしまうことがある。
( まぁ別にそのためだけに用意されているタグではないが )
<form> から </form> までの間に囲まれた部分に含まれる、かつ、
後述する input タグの値を送信することが出来る。

input - ユーザーの入力を受け付けるタグ

ユーザーからの入力を受け付ける。
ボタンだったりチェックボックスだったりその形式は様々。
選択入力をする <select> なんてタグもあるが、
入力情報が欲しいだけなら input タグだけでも実現は出来る。
